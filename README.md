# Online Examination System

## Objective

The objective of this project is to build a simple Online Examination System using Java. The system allows users to attend online exams with secure login and automatic submission functionality.

## Features

* Login using Username and Password
* Update Profile and Password
* Multiple Choice Questions (MCQs)
* Selecting Answers for Questions
* Timer for Examination
* Auto Submit Feature
* Score Display
* Logout Option

## Tools Used

* Java
* VS Code
* JDK

## Steps Performed

1. Created login page using username and password.
2. Added profile and password update functionality.
3. Implemented MCQ-based examination system.
4. Added timer for exam duration.
5. Implemented auto-submit feature after timer completion.
6. Displayed exam result and score.
7. Added logout functionality.

## Outcome

The project was successfully created and tested in Java. It provides a simple and effective way for users to attend exams online with secure login, timed assessment, and automatic submission features.
